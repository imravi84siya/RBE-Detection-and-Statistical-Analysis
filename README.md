# RBE Detection and Statistical Analysis

Python implementation accompanying:

"Automated Detection and Statistical Characterization of Rapid Blue-shifted Excursions in Hα and Ca II 8542 Å Observations"

Authors:
Ravi Chaurasiya
Ankala Raja Bayanna
Robertus Erdélyi

Description:
This repository contains the Python code used to:

- Detect Rapid Blue-shifted Excursions (RBEs)
- Measure projected lengths and widths
- Perform statistical distribution fitting
- Generate all figures presented in the manuscript

The implementation is provided as a Jupyter notebook.
